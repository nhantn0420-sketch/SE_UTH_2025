# Roadmap

This roadmap is a lightweight, living plan. Priorities may change based on feedback and course requirements.

## Near-term
- Stabilize core flows (auth, courses, projects, teams)
- Improve automated tests and CI reliability
- Tighten security posture (secrets handling, dependency updates)

## Mid-term
- Performance tuning for dashboards and reporting
- Better observability (structured logging, tracing)
- UX improvements and accessibility

## Long-term
- More AI-assisted features (milestones, feedback insights)
- Scalable real-time collaboration features
