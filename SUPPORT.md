# Support

## Where to get help
- **Questions**: use GitHub Discussions (recommended).
- **Bugs**: open a Bug Report issue using the template.
- **Feature requests**: open a Feature Request issue.

## Security issues
Do not report security vulnerabilities in public issues. See `SECURITY.md`.

## Before opening an issue
- Search existing issues/discussions.
- Include clear steps to reproduce.
- Attach logs/screenshots when helpful (remove secrets).
